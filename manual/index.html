手册

Rainmeter 能在您的桌面上展示可自定义的皮肤，例如内存占用或者是电池电量， RSS 订阅和天气预报等。许多皮肤甚至有着更加复杂的功能:它们可以记录下您的笔记或是代办清单，启动您所喜欢的软件，或是控制您的媒体播放器 - 这一切都发生在一个干净、简介的界面中，您可以根据您的喜好随时对各个窗口进行重新排序和自定义。 Rainmeter 既是一个软件也是一个工具包，您仅会受到您的想象力和创造力的限制。

入门 Rainmeter

在 下载 和 安装 Rainmeter 后，下一步就是 入门 了。

入门是一个面对 新用户 的关于 Rainmeter 的介绍，包括了如何使用 Rainmeter 以及关于创建和编辑皮肤的入门教程。

对入门章节的阅读将会使您对阅读手册的其余部分以及使用 Rainmeter 的过程感到更加的轻松和愉快

使用 Rainmeter

手册的 用户界面 一节介绍了如何使用本软件在桌面上管理Rainmeter。 使用 管理 界面和 右键菜单 来进行控制, 并使用 关于 来监视重要信息。

不久您就会想要使用在网络上下载的其它皮肤扩展那个仅有 Rainmeter 自带的 illustro 套装的皮肤库。 安装皮肤 一节阐述了应如何安装和加载那些适用于 Rainmeter 的皮肤。

编写和编辑皮肤

Rainmeter 可以直接当做一个皮肤管理器来使用，您可以使用它来加载和使用他人制作的皮肤。但是，只有在你从"使用者"变为"创造者"后，Rainmeter 的真正乐趣和力量才会体现出来。手册的 参考 部分包含了如何制作一个能满足你需要的皮肤的全部所需内容。

设置 包含了有关控制了 Rainmeter 许多重要方面的 Rainmeter.ini 文件的详细信息。

Skins 一节介绍了皮肤的 .ini 文件所包含的内容以及它们的功能。它也能作为皮肤强大特性的一个参考。

下面的 Meters, Measures 和 Plugins 三节中包含了制作一个皮肤的核心内容。Measures 和 plugins 负责从系统和网络中收集信息，而 meters 则使用不同的类型来显示东西，几乎不受样式的限制。

下面的章节为一些有用的皮肤选项和特性提供了相应的内容参考。用 Bangs 和 Mouse Actions 让你的皮肤动起来，同时您也可以使用 Lua Scripting 语言来扩展 Rainmeter 的功能。

Rainmeter 中最让人激动的事就是在社区中的免费且开放的皮肤分享。 在 发布皮肤 一节中了解关于打包和搭建套装和皮肤的相关知识。

更多信息

请务必查看 提示与技巧 一节，以获取 Rainmeter 社区提供的大量指南和建议。

Lua代码片段 一节包含了来自 Rainmeter 团队和用户的有用的Lua代码片段集合。
